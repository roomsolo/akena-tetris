akena

a bare-bones, zero-lag tetris clone built for raspberry pi. powered by pygame.
[⚡] fast · [■] stable · [▲] clean

[>] what’s this?

a confident little tetris remake that doesn’t pretend to be more than it is. tight loop, low footprint, rock-solid on rpi hardware. designed as a calm, predictable focus tool for ptsd users.

[#] features

[■] classic block mechanics, no gimmicks

[π] tuned specifically for raspberry pi

[~] soft, steady pacing (no overload traps)

[λ] pygame core — easy to mod

[*] lightweight, single-file friendly build

[>] run it
```bash
pip install pygame
python3 tetris.py
```

lean. confident. gets the job done without asking permission.
